# Vector
## What is `Vector`

> Vectors are function.
> Mathematically, elements in Vector space.

* Things can be represented with `Vector`
  * Documents
  * Binary Strings
  * Collection of attribute
  * States of a system
  * Probability Distribution
  * and many more
